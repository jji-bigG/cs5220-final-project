# Matrix Selection Criteria (Locked Before Benchmarking)

This project locks matrix selection before performance runs to avoid cherry-picking.

## Selection rules

1. Exactly 12 matrices total.
2. Exactly 4 families, with 3 matrices per family:
   - `irregular_graph`
   - `fem_mesh`
   - `block_structured`
   - `banded_pde`
3. Each family uses a fixed small/medium/large scale progression.
4. Parameter choices are deterministic (fixed in source and fixed RNG seed).
5. No substitution after observing performance.

## Locked synthetic matrix set

- `graph_small`, `graph_medium`, `graph_large`
- `fem_small`, `fem_medium`, `fem_large`
- `block_small`, `block_medium`, `block_large`
- `band_small`, `band_medium`, `band_large`

These are generated by `src/spmv_bench.c` and benchmarked unchanged.
